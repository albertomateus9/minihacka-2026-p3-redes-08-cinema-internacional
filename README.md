# Ingressos de Cinema Internacional

Site estático do projeto **Ingressos de Cinema Internacional**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado). Esta versão funciona como um exemplo genérico e temático do produto, com dados fictícios e uma prévia interativa.

## Ideia

Um site internacional para encontrar ingressos e informações de cinemas, incluindo filme, lugar e avaliação do espaço em diferentes cidades.

## Exemplos temáticos

- **Busca por cidade:** Viajante escolhe Lisboa, Buenos Aires ou Toronto para ver sessões próximas.
- **Comparação de salas:** O site mostra horário, idioma, preço fictício e avaliação do espaço.
- **Compra guiada:** A pessoa escolhe filme, lugar e sessão antes de ir para o ingresso.

# Ingressos de Cinema Internacional

Site estático do projeto **Ingressos de Cinema Internacional**, desenvolvido para a MiniHacka 2026-1 da turma do 3º período de Redes de Computadores (Integrado).

## Ideia

Um site internacional para encontrar ingressos e informações de cinemas, incluindo filme, lugar e avaliação do espaço em diferentes cidades.
